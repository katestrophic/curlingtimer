### Git Branching
|`git branch <NAME>`|Description|
|-|-|
|`main`| main branch, merge into when milestone is complete, stable|
|`dev`| primary development branch, merge into when feature is complete, unstable |
|`feature`| individual feature development, working branch (work in progress)


### Git Commenting
|Symbol|Description|
|------|-----------|
| `⦿` | Normal Commit Node |
| `⩤` | Merge Branch |
| `⩥` | New Branch |
| `※` | Note (Empty Commit) |
| `⟁` | Problem or Concern |
